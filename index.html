FFmpeg-API

API de conversion audio haute performance avec Docker, FastAPI et FFmpeg. Simple, rapide, fiable.

Docker Ready FastAPI FFmpeg Open Source

Fonctionnalités

Multi-Format

Conversion entre 6 formats audio populaires : MP3, OGG, WAV, FLAC, AAC, M4A avec paramètres optimisés.

Hash MD5

Vérification d'intégrité des fichiers pour des téléchargements fiables et sécurisé.

Monitoring

Tableaux de bord et statistiques en temps réel pour suivre les conversions.

Auto-Cleanup

Nettoyage automatique des fichiers après 12 heures pour optimiser l'espace.

Swagger UI

Documentation interactive et testable directement dans le navigateur.

Léger & Rapide

Image Docker optimisée (~630MB) avec conversion asynchrone non-bloquante.

Installation

terminal
git clone https://github.com/leknoppix/ffmpeg-api.git
cd ffmpeg-api
docker-compose up -d --build

# API disponible sur http://localhost:8000
# Swagger UI sur http://localhost:8000/docs
# GitHub Pages: https://leknoppix.github.io/ffmpeg-api

API Endpoints

POST /convert/upload/{input_format}?output_format={fmt}
GET /convert/{job_id}
GET /convert/{job_id}/download
POST /convert/{job_id}/download-delete
DEL /convert/{job_id}
GET /convert/metrics

Déploiement Docker

docker-compose.yml
services:
  converter:
    image: leknoppix/ffmpeg-api:latest
    container_name: audio-converter
    ports:
      - "8000:8000"
    volumes:
      - ./tmp:/tmp/convert
    restart: unless-stopped
commandes
docker pull leknoppix/ffmpeg-api:latest
docker run -d -p 8000:8000 \
  -v $(pwd)/tmp:/tmp/convert \
  --name audio-converter \
  leknoppix/ffmpeg-api:latest
