FFmpeg-API

API de conversion audio avec Docker, FastAPI et FFmpeg

Docker FastAPI FFmpeg Python

Fonctionnalités

Multi-Format

Conversion entre 6 formats audio : MP3, OGG, WAV, FLAC, AAC, M4A

Hash MD5

Vérification d'intégrité des fichiers pour des téléchargements fiables

Monitoring

Statistiques en temps réel via l'endpoint /metrics

Auto-Cleanup

Suppression automatique des fichiers après 12h

Swagger UI

Documentation interactive et testable sur /docs

Léger

Image Docker ~630MB avec tout inclus

Formats Supportés

MP3 - libmp3lame (~192 kbps)
OGG - libvorbis (~96 kbps)
WAV - PCM 16-bit lossless
FLAC - FLAC lossless
AAC - AAC (~128 kbps)
M4A - AAC-LC (~128 kbps)

Installation

git clone https://github.com/leknoppix/ffmpeg-api.git
cd ffmpeg-api
docker-compose up -d --build
# Accès API: http://localhost:8000
# Swagger: http://localhost:8000/docs

API Endpoints

POST /convert/upload/{input_format}?output_format={fmt}
GET /convert/{job_id}
GET /convert/{job_id}/download
POST /convert/{job_id}/download-delete
DEL /convert/{job_id}
GET /convert/metrics

Utilisation Rapide

# Upload et conversion
curl -X POST "http://localhost:8000/convert/upload/mp3?output_format=ogg" \
  -F "file=@audio.mp3"

# Réponse:
# {"job_id": "abc123...", "input_hash": "...", "message": "..."}

# Vérifier le statut
curl http://localhost:8000/convert/{job_id}

# Télécharger
curl -O http://localhost:8000/convert/{job_id}/download

# Télécharger + supprimer avec vérification hash
HASH=$(md5sum downloaded.ogg | awk '{print $1}')
curl -X POST "http://localhost:8000/convert/{job_id}/download-delete" \
  -H "X-Content-Hash: $HASH"
